A very simple plugin for taking notes on your dahsboard.

== Description ==
If you want to be able to write notes easily and quickly on the Dashboard this plugin is for you. Easy Notes is a simple and minimalistic notes plugin which makes life easier for webmasters and coders. No bells and whistles – only a simple and elegant notepad working straight out of the box. 

Features
-	Simple and minimalistic. 
-	Works out of the box. 
-	No settings to configure.
-	Data saved in your WordPress database. 
-	Accessible via the dashboard. 

Do not hesitate to give feedback or request features! 


== Installation ==

1. Upload the easynotes folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
